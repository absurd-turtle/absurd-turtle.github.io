is a web application which represents relational data as an interactive graph. It's main purpose is to help businesses and consultants to understand and make use of their data in a meaningful way. It can be used to restructure the data, find outliers, create all kind of graphs, project the data to a geographical map, etc.

open-source

out-of-tune

is a web application where music enthusiasts can navigate the vast amount of music data in an intuitive way. The purpose is to break out of the bubble algorithms may have created for you and experience new music in an approachable way. A network graph is used to visualize genres, artists, albums and songs and the corresponding connections between them. The user can interact with the graph to expand more nodes or reduce the amount of nodes. One can either start out with the genre overview or dive into ones favorite playlist and start exploring from there. The application was realized with Vue, GraphQL, nodeJS and ArangoDB. The software is free and open source and was published on github. https://github.com/out-of-tune/OOT
